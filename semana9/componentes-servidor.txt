Sistema operativo:
Administra el hardware y los recursos del computador.
 Permite ejecutar programas y servicios necesarios para el funcionamiento del servidor.

Node.js:
Es un entorno de ejecución que permite correr código
 JavaScript fuera del navegador. Se utiliza para crear aplicaciones y servidores.

Aplicación server.js:
Es el archivo que contiene la lógica del servidor. 
Define cómo responder a las solicitudes de los usuarios.

Puerto:
Es un punto de comunicación por el que el servidor recibe solicitudes. 
Permite acceder a diferentes servicios dentro del sistema.

Logs:
Son registros de eventos y actividades del sistema.
 Ayudan a monitorear el funcionamiento y detectar errores.
